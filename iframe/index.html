Poste Tunisienne - Exemple Chatbot Iframe

Cet exemple montre comment intégrer le chatbot dans une page web en utilisant le script "chatbot.js"

Pour intégrer le chatbot dans votre page web, ajoutez le script suivant dans le body de votre page: 

        <script src="chatbot.js" chatbot-url="http://localhost:8000"></script>

Remplacez "http://localhost:8000" par l'URL de votre serveur chatbot
